# UI 设计说明（高保真）

日期：2026-02-07  
执行者：Codex

## 设计原则（iOS HIG 对齐）
- 触控目标最小 44x44pt（https://developer.apple.com/design/human-interface-guidelines/accessibility）
- 文本对比度 ≥ 4.5:1，确保可读性（同上）
- 内容遵循 Safe Area，避免被状态栏/底部指示条遮挡（https://designcode.io/ios-design-handbook-safe-area）
- Tab Bar 建议 3-5 个入口，避免拥挤（https://codershigh.github.io/guidelines/ios/human-interface-guidelines/components/navigation-and-search/tab-bars/）

## 视觉系统
- **字体**：系统字体栈（SF Pro Text/Display）
- **主色**：深灰 #0F172A / #18181B
- **强调色**：靛蓝 #4F46E5 / #6366F1
- **背景**：浅灰 #F1F5F9
- **边框**：#E2E8F0

## 组件与布局
- **卡片**：圆角 16-24，浅灰背景
- **按钮**：主按钮实心，次按钮描边
- **图表**：趋势图与分布图均为柱状表示
- **列表**：左图标 + 标题 + 时间 + 金额
- **弹窗**：底部弹窗，强调操作聚焦
