# 产品界面规划（信息架构）

日期：2026-02-07  
执行者：Codex

## 信息架构总览
- 底部 Tab：记账 / 统计 / 历史 / 设置
- 核心动作：记账入口（语音/手动）必须始终可达

## 页面层级与结构

### 1. 记账（主入口）
- 顶部：本月支出概览
- 中部：语音输入 / 手动输入
- 底部：今日记录列表
- 弹层：手动输入表单（金额、标题、时间、分类）

### 2. 统计
- 顶部：时间维度切换（月/季度/年）
- 中部：关键指标卡（总支出、日均、最大单笔）
- 下部：趋势图（时间序列） + 分布图（分类降序）

### 3. 历史
- 按天分组展示历史记录
- 每组包含当日汇总与列表

### 4. 设置
- 货币单位选择
- 分类管理（增删改查）
- 数据管理（导出 CSV / 清空数据）

## 导航与关键路径
- 首次进入：记账页为默认 Tab
- 关键路径：
  - 记账 → 录入 → 今日记录可见
  - 统计 → 维度切换 → 趋势/分布联动查看
  - 历史 → 日期分组 → 详情回溯
  - 设置 → 分类/货币/数据操作
